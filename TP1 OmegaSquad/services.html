Desarrollo de software

Acá comparto todo lo relacionado con la creación de software, desde la idea hasta que se convierte en una aplicación real que funciona. No se trata solo de programar, sino también de pensar soluciones, probar, equivocarse y volver a intentar hasta que todo encaje. Ya sea una app, una herramienta web o un sistema más complejo, el objetivo siempre es el mismo: construir algo útil, funcional y pensado para quienes lo van a usar.
